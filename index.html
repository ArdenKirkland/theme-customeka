Customize: an Omeka Theme

This is a very customizable Omeka theme that allows you to change the fonts, colors, and background images that make up your Omeka site. In the theme's settings (Admin -> Appearance -> Themes -> Configure), you can set custom values for:

  • Header font and font sizes
  • Header background image
  • Body font and font sizes
  • Body background image
  • Colors

Colors

By changing the two main color settings, Color One and Color Two, you can colorize your entire site. Customize dynamically generates colors for your whole site based on these two colors.
